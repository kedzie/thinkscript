### RedDogReversal

[Add to TOS](https://tos.mx/hLFjcVl)

Alerts and bubbles for Red Dog Reversal (RDR) Buy/Sell on intraday chart.  Optionally plots prior day High/Low/Close and alerts on crosses of those prior levels.
